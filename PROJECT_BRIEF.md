| :--- | :--- |

| **Developer** | **NDAI** (ndai.pro) — Minot, ND |
| **Status** | Phase 1 Development |

## The Problem
SandPro's leadership assigns objectives across departments, but tracking is scattered across email, spreadsheets, and verbal commitments. There's no single system to see what's assigned, what's overdue, who owns it, or whether measurable goals are trending in the right direction. Leadership needs clean data flowing into Power BI for executive reporting.

## The Solution
**SP OMP** — a focused, SandPro-branded web application that gives leaders one dashboard to assign, track, and close objectives with full accountability. Not a project management tool. Not an EOS platform. An execution tracker built for how SandPro actually works.

## Core Capabilities (Release 1)
* **Objective Lifecycle**: Creation, assignment, and status management.
* **Subtasks**: Roll-up progress tracking from child tasks.
* **Metric Tracking**: Baseline → Target → Recurring check-ins.
* **Dashboards**: Role-based views (Executive, Manager, Individual).
* **Notifications**: Automated email reminders (Assignment, Due Soon, Overdue).
* **Power BI Ready**: Clean relational data via API and SQL.
* **Industrial UI**: Orange/Gunmetal aesthetic with zero learning curve.

## Success Criteria
Users can create, assign, update, and close objectives from desktop or mobile. Dashboards show overdue items, workload distribution, and status breakdowns. Data exports cleanly to Power BI. Email reminders actually get sent on time.
